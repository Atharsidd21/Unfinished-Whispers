Thanks for downloading!

These sprites were made to help kickstart the Asset Alliance project and encourage collaboration among indie/hobbyist devs/artists.

I first shared these for my Patrons and I'm releasing them free now to the dev community on itch.io.

This is a full collection of new character sprites in a modern-chibi-pokémon style, with the full four-directional walk animations.

There's eight males and eight females, for a total of 16 characters. 
They're arranged in a sprite-sheet format that will work in RPGMaker, and the frames are 16x32 (2-tile-height), 
aligned to a regular grid so you can easily splice or arrange these sprites with snap-to-grid.

I hope these are useful for you.

-------

If you want to build your own characters in this style, I also included the base that I made for them (in 100% no-zoom so you can paste it into a working file).
Note that it doesn't have animation because with this small scale it's more efficient to animate the full character after using the base only to design the idles.
